Project Sprouts and FlashSDK

This documentation covers the FlashSDK which is a collection of automated build tasks, tools, generators and libraries to support the creation of SWF and AIR content using ActionScript 3 and/or the Flex framework.

This documentation also covers the underlying, general purpose infrastructure that helped create those tools – that is Project Sprouts.

FlashSDK

If you’re working as a Flash developer that is looking for a better way to create Flash content, you’ll be most interested in the documentation within the FlashSDK module.

Project Sprouts

If you’re a developer that wants to get a better understanding of how these tools work, you should start by checking out the FlashSDK module and then digging into the classes found in the Sprout module.
